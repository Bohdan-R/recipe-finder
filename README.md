# Recipe Finder App

## Quick Start

```bash
# Install dependencies
npm install

# Run the development server
npm run dev

## Overview

Recipe Finder is a web application for searching recipes using the Spoonacular API.
Users can search recipes by name, filter by cuisine, and preparation time.

---

## Features

- Recipe search using the Spoonacular API
- Filtering by cuisine and maximum preparation time
- Display of recipe cards with images and ingredient lists
- Responsive design with Tailwind CSS
- Typed with TypeScript
- Proper handling of loading states and errors

---

## Technologies

- Next.js 15 (App Router)
- React 19
- TypeScript 5
- Tailwind CSS 4
- ESLint + Prettier for code quality
- Spoonacular API
```
